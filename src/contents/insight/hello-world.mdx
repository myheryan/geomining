# Selamat Datang di Dunia MDX

MDX memungkinkan Anda menulis konten menggunakan **Markdown** yang digabungkan dengan **Komponen React**. Ini adalah standar baru untuk pembuatan konten di web modern.

<Callout type="info">
  **Catatan:** Komponen Callout ini ditarik secara dinamis dari folder components/mdx. Sangat berguna untuk memberikan highlight informasi penting.
</Callout>

## Mengapa Menggunakan MDX?

Ada banyak alasan mengapa tim engineering besar beralih ke MDX:

1. **Fleksibilitas:** Masukkan grafik, formulir, atau widget interaktif langsung ke dalam tulisan.
2. **Type Safety:** Dengan TypeScript, metadata Anda selalu tervalidasi.
3. **Developer Experience:** Menulis dokumentasi terasa seperti menulis kode.
